# 이커머스 고객 세분화 — CV 기반 RFM 분석

> Dacon 공모전 · 개인 분석 · 2024.09 · 1개월

브라질 이커머스 플랫폼 Olist 데이터를 활용해, 단순 RFM이 아닌 **변동계수(CV) 기반 가중치를 적용한 커스텀 RFM 점수**로 고객을 세분화하고 세그먼트별 마케팅 액션 플랜을 도출한 프로젝트입니다.

---

## 문제 정의

기존 RFM 분석은 R·F·M 세 지표에 동일한 가중치를 부여합니다.  
그러나 비즈니스 특성에 따라 각 지표의 중요도는 다르며, 동일 취급 시 비즈니스 목표와 맞지 않는 세분화 결과가 나올 수 있습니다.

**→ 데이터 분포에 기반한 가중치 설계로 비즈니스 목표 정렬도를 높이는 것이 목표**

---

## 분석 흐름

```
데이터 수집·전처리
    ↓
RFM 점수 계산 (CV 기반 가중치 적용)
    ↓
KMeans 클러스터링
    ↓
ANOVA 통계 검증
    ↓
5등급 10세그먼트 분류
    ↓
코호트 분석 · 연관규칙 분석
    ↓
세그먼트별 마케팅 액션 플랜 도출
```

---

## 핵심 방법론

### 1. CV 기반 커스텀 RFM 가중치 설계 (Feature Engineering)

군집별 변동계수(CV = 표준편차 / 평균)를 각 지표의 가중치로 활용했습니다.  
변동성이 클수록 군집 간 변별력이 크므로, 세분화에 더 많은 영향을 미치도록 설계했습니다.

| 지표 | 가중치 |
|------|--------|
| Recency (최근성) | 0.289 |
| Frequency (빈도) | 0.313 |
| **Monetary (금액)** | **0.398** |

```
RFM Score = 0.289 × R + 0.313 × F + 0.398 × M
```

Monetary의 변동계수가 가장 높아 세분화에 가장 큰 영향 → 비즈니스 목표(매출 중심)와 자연스럽게 정렬

### 2. KMeans 클러스터링 + ANOVA 통계 검증

- KMeans로 고객 군집화
- ANOVA (p < 0.001) 로 세그먼트 간 유의미한 차이 확인 → 세분화 근거 확보
- **5등급 10세그먼트** 분류

| 등급 | 세그먼트 |
|------|---------|
| VIP | 최우수 고객 |
| 충성 | 재구매 활성 고객 |
| 잠재 | 육성 가능 고객 |
| 신규 | 최근 첫 구매 고객 |
| 이탈위험 | 재구매율 낮은 고객 |

### 3. 코호트 분석

- 월별 재구매율 패턴 파악
- 평균 재구매율: **6~23%** (월별 변동)
- 세그먼트별 재구매 주기 차이 식별

### 4. 연관규칙 분석 (MLxtend Apriori)

- 크로스셀링 가능한 상품 조합 도출
- 세그먼트별 맞춤 액션 플랜 정리

---

## 결과

- CV 기반 가중치 RFM 점수로 **5등급 10세그먼트** 분류 완료
- ANOVA 통계 검증으로 세그먼트 간 유의미한 차이 확보 (p < 0.001)
- 세그먼트별 재구매율·구매 패턴 파악 및 크로스셀링 액션 플랜 도출
