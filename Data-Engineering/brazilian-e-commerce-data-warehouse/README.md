## 1. 프로젝트 개요
- **목표**: 이커머스 데이터를 활용하여 데이터 웨어하우스 설계 및 구축, 복잡한 SQL 쿼리 최적화, 데이터 마트 구성을 목표로 합니다.
- **사용 데이터셋**: Brazilian E-commerce 실제 이커머스 데이터 (Kaggle 제공)

- **핵심 내용**:
    - 데이터 웨어하우스 설계 및 구축
    - 복잡한 SQL 쿼리 최적화 (인덱스, 조인, 서브쿼리 등)
    - Window Function, CTE 활용
    - 데이터 마트 구성
